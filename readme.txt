== Description ==
Easily use Hatch on your Wordpress site. 

== Installation ==

1. Upload the hatch-for-wordpress plugin into your WordPress site, activate it and you're done.
2. Available shortcodes
    * [hatch-showroom id=0]
        * id: is your hatch company_id
    * [hatch-review-feed id=0 column="single" width=0 height=0]
        * id: is your hatch company_id
        * column: single|multiple
        * width: default is 0
        * height: default is 0
